Planner Disciplina is a school/academic subject planner developed for the Mobile Development course at Federal University of São Carlos. The system was built using Android Studio with Kotlin and Jetpack Compose.

## Features

The system provides the following functionalities:
- Add and remove subjects
- Add, edit, and remove fields within subjects

## How to Run

1. Install Android Studio
2. Clone the project:
```
git clone git@github.com:GiovannaVictoria/PlannerDisciplina.git
```
3. Open the project in Android Studio
4. Run the project

## Funcionalidades

O sistema oferece as seguintes funções:
- Adição e remoção de disciplinas
- Adição, alteração e remoção de campos dentro das disciplinas

## Como executar

1. Instalar o Android Studio
2. Clonar o projeto:
```
git clone git@github.com:GiovannaVictoria/PlannerDisciplina.git
```
3. Abrir o projeto no Android Studio
4. Executar o projeto
